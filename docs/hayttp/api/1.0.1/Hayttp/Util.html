static array normalizeHeaders(array $headers)

Normalize an array of headers.

Turns this:

[ 'HTTP/1.0 200 OK', 'Content-Type ' => ' application/json', 'x-foo-bar: thing', 'x-baz-bing: ', ]

into this: [ 'HTTP/1.0 200 OK', 'content-type' => 'application/json', 'x-foo-bar' => 'thing', 'x-foo-bing' => '', ]
