static Hayttp mockEndpoint(string $methodPattern, string $urlPattern, callable $handler)

Add a mocked end point to all requests created by the global Hayttp facade.

Mocking end points does not necessarily mean that all HTTP calls will be intercepted. In order to mock all end points, you must add a catch-all end point. Like mockEndpoint( '.*', // match any method '{anything}', // match any URL $handler )

Parameters

string $methodPattern a regular expression to match the method(s) of the call
string $urlPattern A url "pattern". For instance {protocol}://example.{tld}/foo/{dir1}/{dir2}
callable $handler A callable that returns a ResponseContract object

Return Value
