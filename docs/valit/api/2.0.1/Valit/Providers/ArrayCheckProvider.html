AssertionResult checkRelativeCount(mixed $value, string|int $operator, int $against = null)

Check if $value is an array (or countable) where the count compares to $against using the $operator.

Examples:

// count > 20
Check::that($foo)->arrayWhereCount('>', 20);

// count <= 255
Check::that($foo)->arrayWhereCount('<=', 255);

// alternative:
Check::that($foo)->arrayWhereCount('≤', 255);

// check count = 40
Check::that($foo)->arrayWhereCount('=', 40);
Check::that($foo)->arrayWhereCount(40);     // alt. syntax

Parameters

mixed $value The inspected variable
string|int $operator Must be one of >, <, =, >=, <=, ≥, ≤
int $against The count we should compare to.
