AssertionResult checkDateParsable(mixed $value, string|null $format = null)

Check if $value is a string containing a parseable date.

The $format parameter can be null or a string. If it is null, we will attempt to parse the $value as a DateTime via PHPs built-in inference. If $format is a string, it will be used to define the format of the $value. See http://php.net/manual/datetime.createfromformat.php for more info.

Check if $value is a date after $against.

AssertionResult checkBirthday(mixed $value, mixed $otherDate)

Check if $value is a datetime where the day-month-component is the same as the day-month-component of $otherDate.

In other words, are the two dates "birthday-equivalent" of each other.

For instance: '1987-12-01 23:30:00' is birthday equivalent of '1950-12-01 11:32:34' because they both occur in on december 1st.
