protected object; expandedValue(mixed $object)

Expand the data available on an object.

If an object has jsonSerialize() method, all the data returned by that method is appended to the object as public properties. If an object has a __debugInfo() method, all the data returned by that method is appended to the object as public properties, overwriting any properties set by jsonSerialize. If the object has a validationData() method, all data returned by that method is appended to the object as public properties, overwriting any properties set by the previous two method calls.
