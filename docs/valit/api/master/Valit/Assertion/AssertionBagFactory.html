protected createAssertionBag(string|array|AssertionBag|Logic $assertions)

Normalize a set of assertions and add it to $this->assertions.

Assertions can be given as a string, an array, a an AssertionBag. When string-encoded, the string contains a number of assertion-expressions separated by ampersands. When associative array, each key=>value pair can either be name => parameters When numeric array, each entry contains a single assertion-expression.

We normalize them into well-behaved arrays of name => parameters.
