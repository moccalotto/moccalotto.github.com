AssertionResult checkRelativeLength(mixed $value, string|int $operator, int $against = null)

Check if $value is a string where the length compares to $against using the $operator.

Examples:

// length > 20
Check::that($foo)->lengthIs('>', 20);

// length <= 255
Check::that($foo)->lengthIs('<=', 255);

// alternative:
Check::that($foo)->lengthIs('≤', 255);

// check length = 40
Check::that($foo)->lengthIs('=', 40);
Check::that($foo)->lengthIs(40);     // alt. syntax

Parameters

mixed $value The inspected variable
string|int $operator Must be one of >, <, =, >=, <=, ≥, ≤
int $against The length we should compare to.
