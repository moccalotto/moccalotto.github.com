static int toBytes(string $size)

Convert a human-readable size-string to bytes.

For instance: 100 is "converted" to 100 '100kB' is converted to 100.000 '100KiB' is converted to 102.400 '100GB' is converted to 100,000,000,000 '100GiB' is converted to 107.374.182.400
