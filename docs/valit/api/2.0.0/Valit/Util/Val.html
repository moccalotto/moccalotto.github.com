static bool is(mixed $value, string|string[] $types)

Check if a value has a given type or class.

$types can be a string with a type name such as: 'int', 'float', 'bool', 'array' or even pseudy types such as 'callable', 'iterable', 'countable', 'stringable' and 'arrayable'.

It can also be a fully qualified class name such as 'Valit\Check'

It can also be an array of the above types. If it is an array, then $value can be any of the given values. Example: ['int', 'DateTimeInterface']

It can also be a string with many types separated by a pipe | characer. Example: 'string|DateTimeInterface', 'int | float'

Parameters

mixed $value The value to check
string|string[] $types Value must have at least one of the declared types

Return Value

bool

Exceptions

InvalidArgumentException if $types is not a string or an array of strings

static array map(mixed $iterable, mixed $callable)

Map an iterable variable.

Example:

// the two lines below are equivalent:
$escaped = Val::map($array, '::escape');
$escaped = Val::map($array, 'Valit\Util\Val::escape');

$integers = Val::map($array, 'intval');  // php version
$integers = Val::map($array, '::toInt'); // robust version

Parameters

mixed $iterable The array (or traversable) to be mapped.
mixed $callable A callable or a string starting with two colons. If it is a string with two colons, it is actually a short-hand for calling a static function in this class. For instance, if $callable is '::escape' then it is the same as if $callable was 'Valit\Util\Val::escape'.
